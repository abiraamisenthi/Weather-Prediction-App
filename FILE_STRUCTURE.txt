KEY FILES DESCRIPTION:
======================

Frontend Components:
- SearchBar.js: Handles city search input and geolocation
- WeatherDisplay.js: Shows current weather with details
- ForecastDisplay.js: Displays 5-day forecast cards
- SearchHistory.js: Lists recent weather searches

Backend Files:
- server.js: Express server configuration and routes
- weather.js: Weather API endpoints (current, forecast, coordinates)
- searchHistory.js: History CRUD operations
- SearchHistory.js: MongoDB model for search records

Styling:
- App.css: Global styles, gradients, animations
- Component CSS files: Specific styling for each component

Configuration:
- package.json (client): React dependencies and scripts
- package.json (server): Express dependencies and scripts
- .env: Environment variables (API key, MongoDB URI, port)
